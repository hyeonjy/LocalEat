## 로컬잇 (LocalEat)

> **"지역 식당 도장깨기 플랫폼"**  
> 지역 기반 맛집을 탐방하고 기록하며, 사용자들이 리뷰, 스탬프 미션, 커뮤니티를 통해 함께 로컬 맛집 문화를 만들어가는 플랫폼입니다.

<br />

## ✨ 개요
LocalEat은 사용자가 직접 지역 맛집을 탐방하고, 리뷰를 남기며, 스탬프를 통해 도장깨기를 하는 **참여형 지역 맛집 플랫폼**입니다.

<br />

## 📈 핵심 기능 요약
- **회원 관리**: 소셜 로그인, 일반 로그인, 온보딩(지역 설정)
- **홈 화면**: 인기 식당/스토리, 키워드 검색, 지도 기반 맛집 탐색
- **검색 기능**: 키워드 + 지역 조합 검색, 상세 페이지 이동
- **식당 상세**: 리뷰 확인 및 등록(텍스트+이미지, 별점), 키워드 필터
- **커뮤니티**: 게시글/댓글 작성, 우수 작성자 뱃지
- **마이페이지**: 리뷰 관리, 스탬프 기록, 아카이빙 달력/맵, 북마크
